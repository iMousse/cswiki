Why：为什么搜？

- konw sth：知道
- learn sth：学习
- create sth：创作
- do sth：完工



搜什么和为什么搜之间不是一一对应的关系。

为了知道陕西信达罢赛这个事件，你搜了一些相关的信息资讯。关键词：**陕西信达罢赛**

![2551701872475_.pic](assets/2551701872475_.pic.jpg)

里面提到了两个概念：NBL 和 5罚1掷。

我们不知道这两个概念，所以要再去搜一些相关的知识点。什么是NBL？和CBA有什么区别和联系？

5罚1掷什么意思？我们为了了解陕西信达罢赛这个事件，我们需要搜索的内容。有信息资讯也有知识点。



同样的道理，我们为了学习一项技能，我们既要搜索一些知识技能，又需要一些素材文件，甚至一些工具软件。比如我们为了学习ChatGPT，我们要搜索一些视频教程，还可能搜一些提升效率的浏览器插件。

学视频剪辑也是相同的道理，需要搜教程，搜素材，还需要搜一些素材的下载工具。所以当我们要学习一项技能，我们需要搜索很多东西



创作一些东西的时候也是这样，比如要做PPT。我们要需要搜索知识点，一些PPT上的素材文件，帮助我们提升PPT工作效率的插件，比如iSlide。



对于我们要做的一些事情，我们直接搜相关的工具软件就可以了。



这就是搜什么和为什么搜之间的逻辑关系。

结论：从搜索信息资讯的效率来说

谷歌 > 公众号 > 短视频 > 百度

公众号大于短视频是因为博主只管吸引眼球不管准确度，并且短视频创作门槛比较低。

短视频的搜索效率大于百度是因为最近10年，主要的内容生态都沉淀在腾讯和抖音。百度百科和百度知道都是十年前的内容。

谷歌是全世界公认最强，因为谷歌会尽可能直接给你最准确的答案。

比如：搜索杭州亚运会中国获得了多少枚金牌。
